gTTS
=====

:mod:`gTTS <gTTS>` (for *Google text-to-speech*): Python library (and ``gtts-cli`` command-line tool) to use Google Translate's text-to-speech API and write spoken text in ``mp3`` format. It allows unlimited text lengths to be spoken by tokenizing long sentences where the speech would naturally pause. 

Installation
------------

.. code-block:: bash

   pip install gTTS
